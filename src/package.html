What’s next

You will need to convert the package zip to an .appx file to submit it to the Microsoft Store.

Call the CloudAppX web service for this. For instance, using curl: curl -o name.appx -F "xml=@uwp-pkg.zip" https://cloudappx.azurewebsites.net/v3/build This results in a file named name.appx, downloaded in the current directory. Use any name you want.
The input zip must be named uwp-pkg.zip.

You can also do the conversion offline, using the MakeAppx.exe & SignTool.exe utilities that are included in the Windows 10 SDK.
